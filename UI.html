FAQ

  1. Bagaimana cara memastikan tampilan halaman web pada monitor utama dan monitor eksternal tidak tertukar?

    2. Atur monitor utama Anda di Display Settings. Pilih monitor utama Anda pada pilihan monitor. Jika tidak tersedia berarti tidak ada monitor eksternal yang terhubung.

    Scroll ke bawah sampai menemukan checkbox "Mark as Main Monitor" dan centang opsi tersebut.

  2. Mengapa perubahan saya tidak diterapkan?

    3. Setiap perubahan Anda diterapkan ketika Anda memulai ulang software (Restart Software).
